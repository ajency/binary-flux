Import events from files on Amazon S3

Alooma can import files from an Amazon S3 Bucket

  • The content of the files on S3 must include one events per line
  • Events maybe in any text format JSON, csv, syslog, etc (Use a transform function in the Code Engine to parse the events into JSON format)
  • Files may be packaged and compressed with Tar & Gzip
  • You may choose to import all files from the bucket , or only new files created from now on
